# 十一、动画设计原则

所有动画遵守：

短

轻

有反馈

禁止：

长动画

复杂转场

炫技

目标：

每一次点击都有回应。

---

# 十二、情绪设计

产品应该像一个长期陪伴者。

成功时：

给予鼓励。

失败时：

给予包容。

例如：

今天没有完成。

提示：

今天实验室没有继续建设。

不过，

已经完成的旅程不会消失。

明天，

它还在等你。

---

# 十三、UI设计原则

原则一：

梦想永远可见。

原则二：

成长比任务重要。

原则三：

今天比未来重要。

原则四：

允许失败。

原则五：

每一次打卡都值得庆祝。

# 十五、UI一句话总结

Learning OS 的UI，不应该让用户感觉：

"我还有很多任务没做。"

而应该让用户感觉：

"今天，我又把未来建设了一点。"
