你的任务是：为一个名为 Learning OS 的项目，组建一个完整的AI开发团队，并以团队协作方式推进产品开发。

---

# 一、项目背景

Learning OS 是一个 AI无关的学习执行系统，它的核心功能是：

- 导入外部AI生成的 Markdown 学习计划（DSL）
- 解析为结构化任务系统
- 提供任务执行、打卡、XP成长系统
- 提供梦想奖励进度系统

注意：
❗ 不需要你负责生成学习计划
❗ 只负责“执行系统”的开发

---

# 二、你的任务

请你创建一个完整的虚拟开发团队，并满足以下要求：

---

## 1. 团队结构（必须包含）

请生成以下角色：

### 产品层
- 产品经理（PM）
- UX/UI设计师

### 技术层
- 后端工程师（FastAPI / Python）
- 前端工程师（UniApp / Vue）
- 架构师（System Architect）

### 核心引擎层（非常重要）
- DSL Parser工程师（负责Markdown解析）
- Task Engine工程师（负责任务状态机）
- Reward System工程师（XP + 梦想系统）

### 测试与质量
- QA工程师（测试任务流完整性）

### 可选增强
- DevOps工程师（部署与CI/CD）

## 2. 每个角色必须输出

- 所有模块必须可以独立测试
- Parser 和 Task Engine 是最高优先级（P0）
